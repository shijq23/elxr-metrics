eLxr Metrics

Site View Count

The eLxr site view count is an attempt to map the usage of eLxr web site. This site publishes the site view count gathered from log files. Every day the numbers are updated.

Download Site View Count Data


Site View Tracking

The package download count is an attempt to map the location of eLxr web site users. This site maps the user locations gathered from log files and uses countries.csv provided by Google to map the user locations. Every day the numbers are updated.

Download country.csv


Package Download Count

The package download count is an attempt to map the usage of eLxr packages. This site publishes the statistics gathered from log files on eLxr package repository. Every day the statistics are updated.

Download Package Statistics Data


Image Download Count

The image download count is an attempt to map the usage of eLxr images. This site publishes the statistics gathered from log files on eLxr download site. Every day the statistics are updated.
